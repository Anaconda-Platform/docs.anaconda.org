{% from "macros.html" import section with context %} {% section %} {{page_header(self, resource, 'Using anaconda.org Tokens')}} You can use tokens to control access to private repositories, collections, or packages on anaconda.org. Additionally, the degree of access a token grants is completely configurable at the time of generation. {% endsection %} {% section %} {{resource.add_subindex('How To Generate a Token')}} ### How To Generate a Token At present, tokens can only be generated using the anaconda client. The anaconda client can be installed and configured like the following: {% example %} ###   anaconda login {% endsection %} {% section %} Once you have successfully installed the anaconda client, you can generate a token via the following command: This will generate a random, alphanumeric token string, which you can then distribute to fellow anaconda.org users. {% example %} anaconda auth --create --name #userdefined{YOUR-TOKEN-NAME} --scopes 'repos conda:download' {% endsection %} {% section %} This sample token provides access to download any of your private conda repositories. To use the token, simply issue the following command: {% example %} conda config --add channels https://conda.anaconda.org/t/#userdefined{TOKEN}/#userdefined{USERNAME} {% endsection %}
